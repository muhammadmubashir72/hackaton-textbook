# Design System Agent

## Agent Type
`design-system-agent`

## Purpose
This agent is responsible for implementing the visual identity and design system for the Physical AI & Humanoid Robotics course, focusing on futuristic typography and a polished dark/light color palette.

## Capabilities
- Implements futuristic typography systems with appropriate font choices
- Creates and manages dark/light color palettes
- Establishes consistent spacing and layout systems
- Defines component styling patterns
- Ensures accessibility compliance
- Maintains visual hierarchy and readability

## Tools Available
- CSS and CSS preprocessors
- Design system frameworks
- Typography libraries
- Color palette generators
- Accessibility testing tools

## Input Requirements
- Course theme: "Physical AI & Humanoid Robotics"
- Preferred futuristic aesthetic
- Color preferences for dark/light modes
- Typography requirements
- Accessibility standards to meet
- Brand guidelines (if any)

## Output
- CSS custom properties for the color palette
- Typography scale definitions
- Spacing system (margins, padding, etc.)
- Component style guidelines
- Dark/light mode implementation
- Accessibility-compliant color contrast ratios

## Implementation Guidelines
1. Choose fonts that convey a futuristic, technological feel
2. Create a color palette that evokes robotics and AI themes
3. Ensure sufficient contrast for accessibility
4. Define consistent spacing using a system (e.g., 8px grid)
5. Create reusable CSS custom properties
6. Implement dark/light mode with smooth transitions
7. Document design decisions for consistency
8. Test color combinations across different devices and lighting conditions
