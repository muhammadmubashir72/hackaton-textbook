# Course-Specific Sub-Agents for Physical AI & Humanoid Robotics

## Overview
These sub-agents are specifically designed to implement the Physical AI & Humanoid Robotics course frontend. Each sub-agent focuses on a specific aspect of the course interface to ensure high-quality implementation of the futuristic learning platform.

## Hero Section Implementation Agent
- **Type**: hero-section-agent
- **Capabilities**: Creates left-aligned content with right-aligned animated robotics illustration, implements course title, tagline, and call-to-action buttons
- **Tools**: React, CSS modules, Framer Motion
- **Use Cases**: Building the main landing page hero section with futuristic design elements

## Design System Agent
- **Type**: design-system-agent
- **Capabilities**: Implements futuristic typography, polished dark/light color palette, consistent styling
- **Tools**: CSS, design systems, typography frameworks
- **Use Cases**: Establishing the visual identity for the Physical AI & Humanoid Robotics course

## Animation Integration Agent
- **Type**: animation-agent
- **Capabilities**: Integrates Framer Motion for animations and hover effects, creates scroll-based animations
- **Tools**: Framer Motion, CSS animations, JavaScript
- **Use Cases**: Adding smooth animations and interactive effects throughout the course interface

## Course Modules Agent
- **Type**: course-modules-agent
- **Capabilities**: Creates overview section for course modules, organizes content hierarchically
- **Tools**: React components, content management, UI frameworks
- **Use Cases**: Displaying the various learning modules in the Physical AI & Humanoid Robotics curriculum

## Hardware & Lab Agent
- **Type**: hardware-lab-agent
- **Capabilities**: Develops hardware and lab setups section, creates visual representations of equipment
- **Tools**: React components, graphics integration, content display
- **Use Cases**: Showcasing the physical components and laboratory requirements

## Capstone Project Agent
- **Type**: capstone-project-agent
- **Capabilities**: Builds capstone project highlights section, creates project showcase layouts
- **Tools**: React components, gallery layouts, content display
- **Use Cases**: Featuring the final project opportunities for students

## Learning Outcomes Agent
- **Type**: learning-outcomes-agent
- **Capabilities**: Designs student benefits and learning outcomes sections, creates achievement displays
- **Tools**: React components, progress tracking, content organization
- **Use Cases**: Presenting what students will achieve by completing the course

## Assessment & Timeline Agent
- **Type**: assessment-timeline-agent
- **Capabilities**: Creates assessments and timeline details section, implements scheduling displays
- **Tools**: React components, calendar systems, progress tracking
- **Use Cases**: Outlining the evaluation methods and course schedule

## Responsive Design Agent
- **Type**: responsive-agent
- **Capabilities**: Ensures responsive design and mobile compatibility, optimizes layouts for different screens
- **Tools**: CSS Grid, Flexbox, responsive frameworks, testing tools
- **Use Cases**: Making the course accessible on all devices

## Component Reusability Agent
- **Type**: reusability-agent
- **Capabilities**: Creates components that can be reused for future course additions, implements modular architecture
- **Tools**: React component architecture, design systems, pattern libraries
- **Use Cases**: Building a scalable system for additional courses

## Sub-Agent Workflow

### Hero Section Implementation
1. Analyze the course title and tagline requirements
2. Design the left-aligned content layout
3. Create the right-aligned animated robotics illustration
4. Implement call-to-action buttons with hover effects
5. Integrate with the overall design system

### Design System Implementation
1. Research futuristic typography options
2. Create color palette with dark/light modes
3. Establish spacing and layout systems
4. Define component styling patterns
5. Ensure accessibility compliance

### Animation Integration
1. Identify elements that need animation
2. Create smooth entrance animations for sections
3. Implement hover effects for interactive elements
4. Add scroll-triggered animations
5. Optimize for performance

### Course Content Organization
1. Structure course modules logically
2. Create visual representations for each section
3. Implement navigation between modules
4. Add progress tracking capabilities
5. Ensure content accessibility

## Best Practices for Course Sub-Agents
- Maintain consistency with the futuristic theme throughout all implementations
- Ensure all animations enhance the learning experience without being distracting
- Prioritize accessibility and inclusive design in all components
- Optimize for performance across different devices and network conditions
- Create modular, reusable components for future course development
- Follow modern web development standards and best practices
- Test thoroughly across different browsers and devices
