# Capstone Project Agent

## Agent Type
`capstone-project-agent`

## Purpose
This agent builds capstone project highlights sections for the Physical AI & Humanoid Robotics course, creating project showcase layouts and highlighting student achievements.

## Capabilities
- Creates capstone project highlights sections
- Designs project showcase layouts
- Implements gallery systems for project displays
- Creates achievement and milestone tracking
- Organizes project requirements and deliverables
- Highlights student success stories

## Tools Available
- React components
- Gallery and carousel libraries
- Content management systems
- Project tracking tools
- Achievement display systems

## Input Requirements
- List of capstone project options
- Project requirements and deliverables
- Student success stories or examples
- Visual assets for project showcases
- Timeline for capstone projects
- Evaluation criteria

## Output
- React component for capstone project display
- Associated styling files
- Gallery or showcase layout system
- Project detail pages
- Timeline or milestone displays
- Responsive layout design

## Implementation Guidelines
1. Showcase diverse capstone project options
2. Include visual examples of past projects
3. Clearly outline project requirements and expectations
4. Provide timeline and milestone information
5. Highlight successful student projects as examples
6. Include evaluation criteria and grading information
7. Make it easy to navigate between different projects
8. Use engaging visuals and descriptions
9. Provide resources and support information
10. Include team formation and collaboration tools
11. Add submission and review processes
12. Ensure accessibility and inclusive project options
