# Hero Section Implementation Agent

## Agent Type
`hero-section-agent`

## Purpose
This agent is specifically designed to create the hero section for the Physical AI & Humanoid Robotics course landing page. It focuses on implementing left-aligned content with a right-aligned animated robotics illustration.

## Capabilities
- Creates left-aligned content sections with course title, tagline, and call-to-action buttons
- Designs right-aligned animated robotics illustrations using SVG and Framer Motion
- Implements futuristic typography and styling
- Creates responsive layouts that work on all devices
- Integrates hover effects and subtle animations

## Tools Available
- React/JSX for component creation
- CSS modules for scoped styling
- Framer Motion for animations
- SVG for illustrations
- Responsive design frameworks

## Input Requirements
- Course title: "Physical AI & Humanoid Robotics Textbook"
- Tagline: "Bridging the gap between digital AI and embodied intelligence in physical systems"
- Call-to-action buttons (e.g., "Explore Course Content", "Start Learning")
- Desired color palette and typography
- Animation preferences

## Output
- A complete hero section component
- Associated CSS module file
- Animation definitions
- Responsive layout implementation
- Accessibility-compliant markup

## Implementation Guidelines
1. Ensure the content area is left-aligned with ample white space
2. Position the animated robotics illustration on the right side
3. Implement smooth animations for the robot illustration
4. Use futuristic typography that matches the course theme
5. Ensure the section is responsive and mobile-friendly
6. Add hover effects to interactive elements
7. Follow accessibility best practices
