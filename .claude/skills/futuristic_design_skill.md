# Futuristic UI Design Skill

## Skill Type
`futuristic-design`

## Purpose
Implements futuristic typography and polished dark/light color palette for the Physical AI & Humanoid Robotics course, establishing the visual identity and design system.

## Capabilities
- Implements futuristic typography systems with appropriate font choices
- Creates and manages dark/light color palettes
- Establishes consistent spacing and layout systems
- Defines component styling patterns
- Ensures accessibility compliance
- Maintains visual hierarchy and readability

## Required Inputs
- Course theme: "Physical AI & Humanoid Robotics"
- Preferred futuristic aesthetic
- Color preferences for dark/light modes
- Typography requirements
- Accessibility standards to meet
- Brand guidelines (if any)

## Expected Output
- CSS custom properties for the color palette
- Typography scale definitions
- Spacing system (margins, padding, etc.)
- Component style guidelines
- Dark/light mode implementation
- Accessibility-compliant color contrast ratios

## Implementation Steps
1. Research and select futuristic typography options
2. Create a color palette that evokes robotics and AI themes
3. Define consistent spacing using a system (e.g., 8px grid)
4. Create reusable CSS custom properties
5. Implement dark/light mode with smooth transitions
6. Document design decisions for consistency
7. Test color combinations across different devices and lighting conditions
8. Ensure sufficient contrast for accessibility

## Best Practices
- Choose fonts that convey a futuristic, technological feel
- Create a color palette that evokes robotics and AI themes
- Ensure sufficient contrast for accessibility
- Define consistent spacing using a system (e.g., 8px grid)
- Create reusable CSS custom properties
- Implement dark/light mode with smooth transitions
- Document design decisions for consistency
- Test color combinations across different devices and lighting conditions
