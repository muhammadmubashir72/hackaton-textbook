# Course Module Display Skill

## Skill Type
`course-modules`

## Purpose
Creates an overview section for course modules in the Physical AI & Humanoid Robotics curriculum, organizing content hierarchically and making it visually appealing.

## Capabilities
- Creates overview sections for course modules
- Organizes content in a logical hierarchy
- Designs visually appealing module displays
- Implements navigation between modules
- Creates progress tracking capabilities
- Ensures content accessibility

## Required Inputs
- List of course modules with descriptions
- Module structure and relationships
- Visual style preferences
- Navigation requirements
- Progress tracking needs
- Accessibility standards

## Expected Output
- React component for course modules display
- Associated styling files
- Navigation system for modules
- Progress tracking implementation
- Accessible markup structure
- Responsive layout design

## Implementation Steps
1. Organize modules in a logical learning sequence
2. Use visual indicators for module status (completed, in-progress, etc.)
3. Implement clear navigation between modules
4. Provide module previews or summaries
5. Include estimated time for each module
6. Add progress tracking and completion indicators
7. Ensure the layout is responsive and mobile-friendly
8. Use consistent styling that matches the overall design

## Best Practices
- Organize modules in a logical learning sequence
- Use visual indicators for module status (completed, in-progress, etc.)
- Implement clear navigation between modules
- Provide module previews or summaries
- Include estimated time for each module
- Add progress tracking and completion indicators
- Ensure the layout is responsive and mobile-friendly
- Use consistent styling that matches the overall design
- Implement filtering or search capabilities if needed
- Follow accessibility standards for navigation and content
