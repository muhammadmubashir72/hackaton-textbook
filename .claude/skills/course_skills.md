# Course-Specific Skills for Physical AI & Humanoid Robotics

## Overview
These skills are specifically designed for implementing the Physical AI & Humanoid Robotics course frontend. They focus on creating an engaging, futuristic learning experience with advanced animations and interactive elements.

## Frontend Development Skills

### Hero Section Creation
- **Skill**: hero-section
- **Description**: Creates a left-aligned content section with right-aligned animated robotics illustration
- **Use Case**: Implementing the main landing page hero section with course title, tagline, and call-to-action buttons

### Futuristic UI Design
- **Skill**: futuristic-design
- **Description**: Implements futuristic typography and polished dark/light color palette
- **Use Case**: Creating the visual identity for the Physical AI & Humanoid Robotics course

### Animation Implementation
- **Skill**: framer-motion-animations
- **Description**: Integrates Framer Motion for smooth animations and hover effects
- **Use Case**: Adding interactive animations throughout the course interface

### Course Module Display
- **Skill**: course-modules
- **Description**: Creates an overview section for course modules
- **Use Case**: Displaying the various learning modules in the Physical AI & Humanoid Robotics curriculum

### Hardware Integration Display
- **Skill**: hardware-section
- **Description**: Develops hardware and lab setups section
- **Use Case**: Showcasing the physical components and laboratory requirements

### Capstone Project Showcase
- **Skill**: capstone-project
- **Description**: Builds capstone project highlights section
- **Use Case**: Featuring the final project opportunities for students

### Learning Outcomes Display
- **Skill**: learning-outcomes
- **Description**: Designs student benefits and learning outcomes sections
- **Use Case**: Presenting what students will achieve by completing the course

### Assessment Timeline
- **Skill**: assessments-timeline
- **Description**: Creates assessments and timeline details section
- **Use Case**: Outlining the evaluation methods and course schedule

## Animation Skills

## Responsive Design Skills

### Component Reusability

## Skill Invocation Process
1. Identify the specific component or feature needed
2. Select the appropriate skill from this documentation
3. Invoke the skill through the Claude interface
4. Review and refine the implementation as needed

## Best Practices for Course Implementation
- Maintain consistency with the futuristic design theme
- Ensure all animations are smooth and not distracting
- Prioritize accessibility in all design decisions
- Test responsiveness across multiple devices
- Keep component reusability in mind for future courses
