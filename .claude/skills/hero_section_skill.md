# Hero Section Creation Skill

## Skill Type
`hero-section`

## Purpose
Creates a left-aligned content section with right-aligned animated robotics illustration, implementing the course title, tagline, and call-to-action buttons for the Physical AI & Humanoid Robotics course.

## Capabilities
- Creates left-aligned content sections with course title, tagline, and call-to-action buttons
- Designs right-aligned animated robotics illustrations using SVG and Framer Motion
- Implements futuristic typography and styling
- Creates responsive layouts that work on all devices
- Integrates hover effects and subtle animations

## Required Inputs
- Course title: "Physical AI & Humanoid Robotics Textbook"
- Tagline: "Bridging the gap between digital AI and embodied intelligence in physical systems"
- Call-to-action buttons (e.g., "Explore Course Content", "Start Learning")
- Desired color palette and typography
- Animation preferences

## Expected Output
- A complete hero section component
- Associated CSS module file
- Animation definitions
- Responsive layout implementation
- Accessibility-compliant markup

## Implementation Steps
1. Create left-aligned content area with course title and tagline
2. Design right-aligned animated robotics illustration
3. Implement call-to-action buttons with hover effects
4. Apply futuristic typography and styling
5. Ensure responsive design for all screen sizes
6. Add accessibility attributes and semantic markup
7. Optimize performance and loading times

## Best Practices
- Ensure the content area is left-aligned with ample white space
- Position the animated robotics illustration on the right side
- Implement smooth animations for the robot illustration
- Use futuristic typography that matches the course theme
- Ensure the section is responsive and mobile-friendly
- Add hover effects to interactive elements
- Follow accessibility best practices
